Componentes

Input

  • Query Box
    • Sugerencias
    • Comandos, site:
    • Comunes, significado de, traducir, etc.
  • Filtros
    • Contexto (Web/Todo, Videos, Imágenes, Noticias, Mapas, Respuestas, Libros, Aplicaciones)
    • Fechas
    • Regiones / Local
    • Safe Search / Adult Content
    • Para imágenes
      • tipo
        • cara, foto, animación, clip art
      • resolución
      • layout
      • colores
      • licencia
    • Para videos
      • resolución
      • duración
      • calidad
    • Para noticias
      • relevancia
      • reciente

Output

  • Total de resultados y tiempo
  • Quizas quiso decir?
    • Typos / Lenguaje
  • Resultados relacionados
  • Knowledge Panel / Card
  • Resultados
    • Título
    • Fecha
    • Resultados similares a este
    • Link
    • Snippet
    • Traducir
    • Cache
    • Información extra
    • Imagen
    • Ícono
  • Paginación
  • Anuncios

Data

  • Universo de páginas
  • Orden de aparición
  • Texto resaltado

Dinámica

  • Paginación, automática o con click

Contexto de recomendaciones

  • Historial
  • Estadísticas
  • MRU, MR searched

Falta o sobra alguno? Algun comentario o sugerencia?
